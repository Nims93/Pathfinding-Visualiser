Pathfinding

A pathfinding algorithm seeks to find the shortest path between 2 nodes. This app visualises this process. Click on the Algorithm menu and select the pathfinding algorithm you want to create a path.

2 / 7

Animation Speed

Click on the animation speed button to toggle between different or no animation speeds. You can click the button to toggle between no animations, fast or slow animations. Fast animations are recommended for the best blance between a nice animation and not having to wait too long for completion.

3 / 7

Maze Generation

The app can find paths around obstructions as long as a path exists between the start and end nodes. Hover over the mazes dropdown menu and click on the maze generation method you want to create a maze of wall nodes. Maze generation is subject to the animation speed toggle.

4 / 7

Wall Drawing

You can also create you own maze by clicking or clicking and dragging to create a continuous wall. Clicking or click dragging over a wall node will also remove it.

5 / 7

Realtime Pathfinding

If you have already clicked and executed a pathfinding algorithm you can now click and drag either the start or end nodes to get a realtime search and path visualization as you go!

6 / 7

Clearing the Board

Click the "Clear Board" button to remove all visied, path and wall nodes. Clearing the board also disables realtime pathfinding until you select another pathfinding algo.

7 / 7

Change Grid Size

Finally, you can change the grid size with the grid size slider . The larger the grid the longer it will potentially take to find a path. The pathfinder may become laggy on larger grid sizes so be careful with the total amount of nodes the pathfinding algorithm will have to search! Happy Pathfinding!
