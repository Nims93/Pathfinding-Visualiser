Pathfinding

Mouseover the Algorithm menu and select the pathfinding algorithm you want to create a path if one exists. A*: Guaranteed shortest path. Greedy Bredth First Search: Path with the very basic direction heuristic, most of the time will find a path with the least number of visited nodes. Bi-directional Breadth First Search: basic search from both start and end node simultaneously. Breadth First Search: basic radial search outward from start node. Depth First Search: searches along the length of a path as long as possible. Very inefficient.

2 / 7

Animation Speed

Click on the animation speed to toggle between speed of pathfinding animations. Fast: the algorithm will animate and end quickly. Slow: the algorithm will animate and end slowly. None: the algorithm will not animate and the path (if one exists) with appear immediately

3 / 7

Maze Generation

Hover over the mazes dropdown menu and click on the maze generation method you want to create a maze. Maze generation is subject to the animation speed toggle.

4 / 7

Wall Drawing

You can also create you own maze by holding down by left click or clicking and dragging to create a continuous wall. Clicking or clikc dragging over a wall node will also remove it.

5 / 7

Realtime Pathfinding

If you have already clicked and executed a pathfinding algorithm you can now click and drag either the start or end nodes to get a realtime search and path as you go.

6 / 7

Clearing the Board

Click the "Clear Board" button to remove all pathfinding and wall nodes. Clearing the board also disables relatime pathfinding.

7 / 7

Change Grid Size

Finally, you cna change the grid size with grid ize slider . The larger the grid the longer it will potentially take to find a path. The pathfinder may become laggy on larger grid sizes so be careful with the total amount of nodes the pathfinding algorithm will have to search (as indicated by the blue nodes).
